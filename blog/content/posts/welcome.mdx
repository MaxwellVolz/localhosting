# Welcome to Your Blog

This is a sample blog post to demonstrate the MDX + Next.js setup.

## Features

- **Static Site Generation**: Fast, SEO-friendly pages
- **MDX Support**: Write JSX in your markdown
- **Decap CMS**: Visual content management at `/admin`
- **Syntax Highlighting**: Beautiful code blocks with copy buttons
- **Heading Anchors**: All headings are linkable

## Custom Shortcuts

You can use custom text replacements:
- Type `->` to get →
- Type `<-` to get ←
- Type `<3` to get ♥

## Keyboard Shortcuts

Use `::text::` to create keyboard shortcuts like ::Ctrl+C:: or ::Cmd+V::.

## Code Example

```javascript
function greet(name) {
  console.log(`Hello, ${name}!`);
}

greet('World');
```

Edit this file at `content/posts/welcome.mdx` or create new posts via the CMS at `/admin`.
